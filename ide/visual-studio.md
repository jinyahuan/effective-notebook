### 快捷键

#### 常用快捷键

| 快捷键(, 代表是组合快捷键)                    | Description | 描述 |
| :---                     						| :---		  | :--- |
||||
| Ctrl+C,Ctrl+V			   || 复制当前行 	|
| Ctrl+E,Ctrl+D			   || 代码格式化	|
| Ctrl+K,Ctrl+C			   || 注释 			|
| Ctrl+K,CTRL+U			   || 取消注释		|
| Ctrl+K,Ctrl+F 		   || 代码格式化	|
| Ctrl+Shift+U			   || 单词大写		|
||||
| Ctrl+J				   || 代码补全 		|
| Ctrl+U				   || 单词小写		|
| Ctrl+Shift+Space 		   || 查看方法重载	|
